## index.json 结构

```json5
[
    {
        "mapID": [586, 587, 588],
        "mapName": "西津渡",
        "npcID": [109570, 111968, 112443],
        "npcName": "张景超",
        "banner": "banner/张景超.jpg"
    },
    // {
    //     "mapID": 地图ID的数组，表示这个boss会在哪些地图里面出现
    //     "mapName": 地图ID名称，表示这个boss出现的地图名称
    //     "npcID": NPC模板ID，表示这个使用该图片资源的NPCID，是一个数组。用于表示不同难度的boss，一般分别对应10人，pt，yx的难度。
    //     "npcName": boss的名字,
    //     "banner": 以该仓库根目录为基准，图片的位置
    //     "icon": NPC头像/图标
    //     "****": 自定义的各种、背景、大图啥的，最好就用****/***.jpg/png
    // }
]
```
